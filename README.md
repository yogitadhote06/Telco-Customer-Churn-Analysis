## 📌 Project Overview

Customer churn is one of the biggest challenges for subscription-based businesses. This project analyzes customer behavior using the IBM Telco Customer Churn dataset and builds machine learning models to predict whether a customer is likely to churn.

The project also includes an interactive **Streamlit web application** that allows users to explore the dataset, visualize insights, and predict churn for new customers.

# 🤖 Machine Learning Models

The following models were trained and compared:

- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting

Evaluation Metrics:

- Accuracy
- Precision
- Recall
- F1 Score

The best-performing model is automatically saved and used by the Streamlit application.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Telco-Customer-Churn-Analysis.git
```

Move inside the project

```bash
cd Telco-Customer-Churn-Analysis
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
